# Usb-Serial

MCP2200 based USB-Serial converter

## Features  
*FTDI/Arduino pinout  
*RTS auto-reset enables  
*3v3 / 5v selectable operating (supply and logic) voltage  
*Supply target switch  
*Drivers no needed    
*Female bottom connector  
*Micro usb connector
